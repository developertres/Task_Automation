All service accounts will be named using the standard of a prefix of the domain they are created in, followed by a short indicator of the Application, and another indicator or the role of the service account. There is a maximum of 20 characters that can be used for the name. For example, a service account to be used for the Hyperion application performing file transfer would be named “CED-HyperionFTP” Please select a service account name that meets this standard, and list it in the Service Account Name field below.
